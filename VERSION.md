# VERSION

이 문서는 `v1.0` 서비스 시작 이후의 주요 기능과 변경 사항을 기록합니다.
앞으로 `v1.x` 범위의 수정과 개선 사항도 계속 누적합니다.

## v1.0

- 정식 서비스 시작
- 사용자 로그인 및 승인 기반 접근 제어 적용
- 요약 보기, 상세 보기, 지출결의서 작성/조회, 사용자 권한 관리 화면 운영 시작
- 보고서 조회 및 기본 업무 흐름 사용 가능한 상태로 배포

## v1.0.1

- 지출결의서 조회 화면의 PC 문서 양식을 실제 사용 양식에 맞춰 보정
- 지출결의서 하단 결재/서명 표시와 비고 영역 레이아웃 정리
- 지출결의서 조회 화면에서 인쇄 기능 추가
- 모바일에서 지출결의서 내용을 읽기 쉽게 전용 카드형 상세 UI 추가
- PC/인쇄용 지출결의서 문서 양식을 검정 계열 기준으로 정리
- 모바일 전용 지출결의서 상세에서 확인 정보 영역을 제거하고 작성자 정보 중심으로 단순화
- 지출결의서 목록과 모바일 상세에 작성자 이름이 표시되도록 보정
- 모바일 전용 지출결의서 상세 UI를 서비스 기본 색상 톤에 맞게 수정

## v1.1.0

- 자유게시판 메뉴 추가 및 로그인한 전체 사용자 대상 게시글 조회/작성 기능 도입
- 자유게시판 게시글 상세 보기, 본인 글 수정/삭제, 마스터의 전체 글 삭제 권한 추가
- Supabase 자유게시판 테이블과 RLS 정책 스크립트 추가

## v1.1.1

- Vercel Web Analytics 패키지 설치 및 전역 Analytics 컴포넌트 연결
- Vercel Speed Insights 패키지 설치 및 전역 Speed Insights 컴포넌트 연결

## v1.1.2

- PC 지출결의서 인쇄 시 본문 다음 페이지부터 첨부 영수증이 함께 출력되도록 확장
- 영수증 이미지 비율에 따라 세로형/가로형/정사각형 기준으로 인쇄 크기를 조절해 종이 낭비를 줄이도록 보정
- 인쇄용 영수증 영역에서 파일명은 제외하고, 이미지 준비가 끝난 뒤 인쇄할 수 있도록 버튼 상태를 보강
