# VERSION

이 문서는 `v1.0` 서비스 오픈 이후의 주요 기능과 변경사항을 기록합니다.
앞으로 `v1.x` 범위의 수정은 이 파일에 계속 누적합니다.

## v1.0

- 정식 서비스 오픈
- 사용자 로그인 및 승인 기반 접근 제어 적용
- 요약 보기, 상세 보기, 지출결의서 작성/조회, 사용자 권한 관리 화면 운영 시작
- 보고서 조회 및 기본 업무 흐름 사용 가능 상태로 배포

## v1.0.1

- 지출결의서 조회 화면 PC 양식을 실제 사용 양식에 맞춰 재정렬
- 지출결의서 하단 결재/서명 표와 비고 영역 레이아웃 보정
- 지출결의서 조회 화면에서 인쇄 가능하도록 출력 기능 추가
- 모바일에서 지출결의서 내용을 읽기 쉽게 전용 카드형 상세 UI 추가
- PC/인쇄용 지출결의서는 문서 양식에 맞춰 검정 계열로 정리
- 모바일 전용 지출결의서 상세에서 확인 정보 영역을 제거하고 작성자 정보 중심으로 단순화
- 지출결의서 목록과 모바일 상세에 작성자 이름을 표시하도록 보정
- 모바일 전용 지출결의서 상세 UI를 서비스 기본 색상 톤에 맞게 재정비

## v1.1.0

- 자유게시판 메뉴 추가 및 승인된 전체 사용자 대상 게시글 조회/작성 기능 도입
- 자유게시판 게시글 상세 보기와 본인 글 수정/삭제, 마스터의 전체 글 삭제 권한 추가
- Supabase 자유게시판 테이블과 RLS 정책 스크립트 추가
